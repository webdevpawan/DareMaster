# DareMasterBot - A Local Telegram Bot

**DareMasterBot** is a Telegram bot that brings the classic Truth or Dare game to your Telegram chats.

## Features

- Play Truth or Dare in your Telegram group chats.
- Easy configuration via a .env file.

## Prerequisites

Before running **DareMasterBot**, ensure you have the following installed:

- Python (3.6+)

## Installation

1. Clone this repository to your local machine.

```bash
git clone https://github.com/yourusername/DareMasterBot.git
```

2. Install the required dependencies using pip:

```bash
pip3 install -r requirements.txt
```

3. Create a .env file in the project directory and add the following variables:

```
BOT_TOKEN=your_bot_token_here
APP_ID=your_app_id_here
API_HASH=your_api_hash_here
```

## Usage
Run the bot on your local server:

```
python main.py
```
